# SikkhaHub 

**SikkhaHub** is a modern and digital tuition management platform designed to create a transparent and secure bridge between qualified mentors and students.

##  Purpose
The main goal of this project is to redefine the tuition ecosystem in Bangladesh. It simplifies the process of finding academic guidance by providing a platform where trust is built through manual verification of tutors. It ensures a professional experience by digitizing tracking and payments for both tutors and parents.

## Features
* **Role-Based Dashboards:** Customized views and tools specifically for Students, Tutors, and Admins.
* **Verified Tutors:** Every tutor profile undergoes a manual review process to ensure high academic standards.
* **Secure Payments:** Integration with **Stripe** for automated, secure, and transparent financial transactions.
* **Tuition Management:** Students can post detailed requirements, and admins manage the approval/rejection of these posts.
* **Application Tracking:** Tutors can apply for posts, and students can review, approve, or reject applications in real-time.
* **Revenue History:** Tutors can track their lifetime earnings and transaction details directly from their dashboard.
* **Digital Class Tracking:** Monitor progress and attendance through personalized dashboards.
* **Responsive UI:** A dark-themed, modern interface built with smooth animations and professional design.

## Packages Used

### Frontend:
* **React:** For building the dynamic user interface.
* **React Router DOM:** For handling navigation and role-based routing.
* **Firebase:** For user authentication (Email/Password and Google Social Login).
* **Framer Motion:** For advanced UI/UX animations.
* **Tailwind CSS & DaisyUI:** For modern, responsive styling and pre-built components.
* **SweetAlert2:** For professional popup notifications and confirmation dialogs.
* **AOS (Animate On Scroll):** For smooth scroll-triggered animations.
* **JWT Decode:** To safely handle and decode role-based access tokens.
* **Recharts:** For visual data representation and admin analytics.

### Backend:
* **Express.js:** For building the RESTful API.
* **MongoDB:** For flexible, document-based data storage.
* **Stripe:** For processing secure online payments.
* **JSON Web Token (JWT):** For secure, token-based authentication and authorization.
* **Firebase Admin SDK:** For enhanced security and server-side verification.
* **Dotenv:** For managing sensitive environment variables.
